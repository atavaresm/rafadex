## Global Constraints

- Spec: `docs/superpowers/specs/2026-07-21-v1.1-polish-design.md` (approved).

- **Displayed names never change.** Only text handed to `Sound.speak()` uses the `speak` override; `mon.name` in the DOM is always the real English name.
- House Python style (STANDARDS): PascalCase classes, lowerCamelCase functions/variables, no other underscores; ruff `line-length=100`; tests named `testXxx`.
- Vanilla JS, no framework/bundler; all URLs relative; UI copy pt-BR, code/comments English.
- Git flow: feature branch off `develop`, PR-merge into `develop`, then a final PR `develop` → `master` (the existing GitHub Actions workflow auto-deploys `master` to Pages — no new deploy setup needed this round).
- Commit trailer: `Co-Authored-By: Claude Fable 5 <noreply@anthropic.com>`.
- Pronunciation data provenance: the 807-entry `pronounce-dex.js` seed was hand-supplied by the parent from real listening against a pt-BR voice (not machine-tested — the agent cannot perceive audio). Each entry was matched to its numeric id by normalizing both the supplied name and every `pokemon.json` name (diacritic-stripping, punctuation-to-hyphen, case-fold) and, for the ~24 multi-form species (e.g. `Deoxys` → `Deoxys-Normal`), falling back to a unique-prefix match. In `build.py`, add this constant and function near `I18N_LINE`/`parseI18nDex` (after `parseI18nDex`, before `buildDexEntries`):

```python
PRONOUNCE_LINE = re.compile(r'^\s*(\d+):\s*"(.*?)",?\s*$')


def parsePronounceDex(text):
    parsed = {}
    for line in text.splitlines():
        match = PRONOUNCE_LINE.match(line)
        if match:
            parsed[int(match.group(1))] = match.group(2)
    return parsed
```

Replace the existing `buildDexEntries` function with:

```python
def buildDexEntries(pokemon, i18n, pronounce=None):
    pronounce = pronounce or {}
    entries = []
    for mon in pokemon:
        override = i18n.get(mon["id"], {})
        entry = {
            "id": mon["id"], "name": mon["name"], "gen": mon["gen"], "types": mon["types"],
            "evo": mon["evolution"]["stages"],
            "cat": override.get("category", mon["category"]),
            "flavor": override.get("flavor", mon["flavorText"]),
            "power": mon["stats"]["total"],
        }
        if mon["id"] in pronounce:
            entry["speak"] = pronounce[mon["id"]]
        entries.append(entry)
    return entries
```

Replace the existing `buildDataset` function with:

```python
def buildDataset(root=POKEDEX_ROOT, outPath=Path("data/dex.js"), pronouncePath=Path("pronounce-dex.js")):
    pokemon = json.loads((root / "data" / "pokemon.json").read_text())
    i18n = parseI18nDex((root / "i18n-dex.js").read_text())
    pronounceText = pronouncePath.read_text() if pronouncePath.exists() else ""
    pronounce = parsePronounceDex(pronounceText)
    entries = buildDexEntries(pokemon, i18n, pronounce)
    outPath.parent.mkdir(parents=True, exist_ok=True)
    outPath.write_text(renderDexJs(entries))
    print(f"dex.js: {len(entries)} entries, {sum(1 for m in pokemon if m['id'] in i18n)} pt-BR, "
          f"{len(pronounce)} pronounce overrides")
    return entries
```

  }
